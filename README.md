# YOLOv8 Drone Object Detection

Studentski projekat iz oblasti vestacke inteligencije, obrade slike i mikrokontrolerskih sistema.

Tema:
Prepoznavanje ljudi, vozila i gradjevina na dron snimcima koriscenjem YOLOv8 modela.

## Cilj projekta

- Implementacija YOLOv8 modela za object detection
- Detekcija ljudi, vozila i gradjevina
- Obrada slika i video snimaka
- Brojanje detektovanih objekata
- Merenje FPS-a i inference time-a
- Priprema modela za Jetson Nano 2GB
- Pokretanje inference dela na Jetson Nano ploci

## Struktura projekta

- docs/ - projektna dokumentacija
- data/ - slike, video snimci i dataset
- models/ - YOLO modeli
- src/ - Python kod
- deployment/jetson/ - skripte i uputstva za Jetson Nano
- outputs/ - rezultati detekcije
- tests/ - osnovni testovi
